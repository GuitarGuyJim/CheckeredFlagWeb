# CheckeredFlagWeb


## Pug Files and Hierarchy:

      layout.pug
          |
          |




- **layout.pug**: This is the base pug file used in all the pug files for the site.  It
            contains the html head section and defines the banner displayed at the top
            of all the pages.  The block content defined at the end is replaced by
            the pug for specific pages.
